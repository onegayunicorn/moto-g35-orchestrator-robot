# Off-the-Shelf Kit Build Guide

Build a physical system using pre-made components.

## Bill of Materials (BOM)
- Arduino Nano RP2040 Connect
- MPU6050 Accelerometer
- DHT22 Temp/Humidity Sensor
- OV5640 Camera Module
- 3D Printed Enclosure

## Assembly Steps
1. **PCB Mounting**: Secure sensors to the custom PCB.
2. **Wiring**: Connect the Arduino to the sensor array.
3. **Enclosure**: Place the assembly into the 3D printed case.
4. **Connection**: Connect to the Moto G35 via USB-C.\n
